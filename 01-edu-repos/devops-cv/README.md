# 01Talent France - Portfolio Cloud DevOps

[![Version](https://img.shields.io/badge/version-1.0.0-green.svg)](https://github.com/01talent/01talent-cv)

## 🌟 Vue d'ensemble

**01Talent France** est une plateforme web interactive dédiée à la présentation de notre promotion exclusive de talents **Cloud DevOps** fraîchement certifiés. Ce portfolio moderne met en lumière l'expertise technique et l'expérience professionnelle de nos développeurs, formés selon notre méthode pédagogique reconnue et déployée dans plus de 40 pays.

## 🚀 Fonctionnalités principales

### ✨ Design moderne et responsif
- **Interface futuriste** avec animations fluides et effets visuels avancés
- **Design adaptatif** optimisé pour desktop, tablette et mobile
- **Animations de révélation** avec gestion intelligente des timings
- **Thème sombre** avec palette bleue distinctive

### 🎯 Navigation interactive
- **Navigation verticale** avec pagination élégante
- **Animations de scroll** fluides entre sections
- **Indicateurs visuels** de progression

### 👥 Portfolio des talents
- **Présentation individuelle** de chaque développeur Cloud DevOps
- **Vérification de disponibilité** en temps réel
- **Accès direct aux CV** via modales intégrées
- **Sélection multiple** pour demandes groupées

### 📊 Informations clés
- **100% de certification** garantie
- **6+ mois de formation intensive**
- **Technologies Cloud & DevOps** de pointe

## 🛠️ Technologies utilisées

### Frontend
- **HTML5** - Structure sémantique et accessible
- **CSS3** - Animations avancées, Flexbox, Grid
- **JavaScript (ES6+)** - Interactions dynamiques

- **CSS Animations** - Transitions fluides et effets visuels

## 🚀 Installation et lancement

### Prérequis
- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Aucun serveur requis - fonctionne directement depuis les fichiers locaux

### Démarrage rapide

1. **Cloner ou télécharger** le projet :
   ```bash
   git clone https://github.com/01talent/01talent-cv.git
   cd 01talent-cv
   ```

2. **Ouvrir dans le navigateur** :
   ```bash
   # Méthode 1 - Ligne de commande
   open index.html

   # Méthode 2 - Interface graphique
   # Double-cliquer sur index.html dans l'explorateur de fichiers
   ```

3. **Profiter de l'expérience** 🎉

## 🎮 Utilisation

### Navigation
- **Scroll vertical** : Naviguer entre les sections
- **Points de pagination** : Accès rapide aux sections
- **Flèches du slider** : Parcourir les descriptions

### Sélection de talents
1. **Explorer les profils** dans la section dédiée
2. **Cliquer sur les checkboxes** pour sélectionner
3. **Consulter les CV** via les boutons intégrés
4. **Valider la sélection** avec le bouton flottant

### Interactions disponibles
- **Swipe mobile** : Glisser entre les slides du hero
- **Drag desktop** : Glisser avec la souris
- **Auto-play** : Défilement automatique (6 secondes)
- **Navigation par flèches** : Contrôle manuel du slider

## 🎨 Personnalisation

### Modification des couleurs
Dans `assets/css/style.css`, modifiez les variables CSS :
```css
:root {
    --primary-blue: #2d64b8;  /* Couleur principale */
}
```

### Ajout de nouveaux talents
1. Ajouter les informations dans `index.html`
2. Suivre le modèle des cartes existantes
3. Mettre à jour les chemins vers les CV

### Modification du contenu
- **Section Hero** : Modifier les slides dans `index.html`
- **Informations société** : Personnaliser les statistiques
- **Descriptions** : Adapter le texte selon les besoins
